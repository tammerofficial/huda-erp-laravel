## ðŸ‘¥ Employee Management

### **Adding New Employees**

#### **Step 1: Basic Information**
1. Navigate to **Human Resources > Employees**
2. Click **"Add New Employee"**
3. Fill in required fields:
   - **Full Name:** Employee's complete name
   - **Employee ID:** Unique identifier
   - **Position:** Job title
   - **Department:** Work department
   - **Hire Date:** Employment start date

#### **Step 2: Salary Information**
- **Salary Type:** Choose from Monthly, Per Piece, or Hourly
- **Base Salary:** Monthly base amount
- **Overtime Rate:** Additional pay for overtime
- **Bonus Rate:** Performance bonus percentage

#### **Step 3: Personal Details**
- **Nationality:** Employee's nationality
- **Civil ID:** Government ID number
- **Passport Number:** International travel document
- **Blood Type:** Medical information
- **Emergency Contact:** Contact person details

#### **Step 4: Document Upload**
- **Profile Photo:** Employee's picture
- **ID Card:** Front and back images
- **Passport:** Passport photo
- **Contract:** Employment contract
- **Medical Certificate:** Health clearance

### **Employee Events Management**

#### **Creating Events**
1. Go to **Employee Profile > Events**
2. Click **"Add Event"**
3. Select event type:
   - **Vacation:** Time off requests
   - **Sick Leave:** Medical absences
   - **Training:** Skill development sessions
   - **Performance Review:** Evaluation meetings
   - **Birthday:** Personal celebrations
   - **Anniversary:** Work milestones

#### **Recurring Events**
- Set events to repeat daily, weekly, monthly, or yearly
- Automatic reminders for recurring events
- Color coding for different event types

## â° Attendance System

### **Manual Check-In/Check-Out**

#### **Individual Check-In**
1. Navigate to **Production > Attendance**
2. Click **"Check In"**
3. Select employee from dropdown
4. Enter check-in time
5. Add notes if needed
6. Click **"Save"**

#### **Bulk Check-In**
1. Go to **Attendance > Bulk Check-In**
2. Select multiple employees
3. Set check-in time for all
4. Click **"Bulk Check-In"**

### **QR Code Attendance**

#### **Employee QR Code**
1. Go to **Employee Profile > QR Code**
2. Click **"Generate QR Code"**
3. Download or print the QR code
4. Employee can scan to check in/out

#### **QR Scanner Usage**
1. Navigate to **Production > QR Scanner**
2. Allow camera access
3. Point camera at employee's QR code
4. System automatically processes check-in/out

### **Attendance Reports**

#### **Monthly Report**
1. Go to **Attendance > Reports**
2. Select month and year
3. Choose report format (PDF/Excel)
4. Click **"Generate Report"**

#### **Individual Employee Report**
1. Go to **Employee Profile > Attendance**
2. Select date range
3. View detailed attendance history
4. Export if needed

## ðŸ­ Production Management

### **Production Orders**

#### **Creating Production Order**
1. Navigate to **Production > Production Orders**
2. Click **"Create New Order"**
3. Fill in order details:
   - **Product:** Select from product list
   - **Quantity:** Number of items to produce
   - **Start Date:** Production start date
   - **Expected Completion:** Target completion date
   - **Priority:** High, Medium, or Low

#### **Production Stages**
- **Cutting:** Fabric cutting stage
- **Sewing:** Stitching operations
- **Embroidery:** Decorative work
- **Ironing:** Pressing and finishing
- **Quality Check:** Final inspection

### **Production Logs**

#### **Recording Production Work**
1. Go to **Production > Production Logs**
2. Click **"Add Log Entry"**
3. Fill in details:
   - **Employee:** Who performed the work
   - **Stage:** Production stage
   - **Product:** Item being produced
   - **Pieces Completed:** Number of items finished
   - **Start Time:** When work began
   - **End Time:** When work finished
   - **Quality Status:** Work quality assessment

#### **Auto-Assignment**
- System automatically assigns best employee for each stage
- Considers employee skills, workload, and efficiency
- Balances work distribution across team

### **Production Tracking**

#### **Real-Time Monitoring**
- View current production status
- Track completion percentages
- Monitor employee efficiency
- Identify bottlenecks

#### **Production Reports**
- Daily production summaries
- Employee performance metrics
- Stage completion times
- Quality statistics
